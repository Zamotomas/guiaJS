Variables y Constantes en JavaScript

Entre los tipos de variables tenemos 3

        const nombreEmpleado = "José Perez";   // Genera una Variable que no se puede modificar en el futuro
        let direccion = "Av.105 - Calle 3";    // Genera una variable que solo vive en el bloque donde se creo
        var costoDelPasaje = 1500;             // Genera una varible general donde puede ser llamada de cualquier 
        let tasaEmbarque = 50.5;                  otro bloque, estas no se recomiendan usar 
        var Variable1, Variable2;              // Podemos Inicializar muchas variables a la vez

Estas variables pueden ser llamadas de muchos modos entre los cuales estan estos:

        const nombrePasajero = "Alan"
        const apellidoPasajero = "Peña"

        let nombreCompleto = `El nombre completo es: ${nombrePasajero} ${apellidoPasajero}`;

Operadores Aritmeticos y Logicos

Operadores Aritmeticos

Entre los operadores Aritmeticos tenemos muchas abreviaciones

          +=      b += 3  equivale a  b=b+3
          -=      b -= 3  equivale a  b=b-3
          *=      b *= 3  equivale a  b=b*3
          /=      b /= 3  equivale a  b=b/3
          %=      b %= 3  equivale a  b=b%3
          ++      b++     equivale a  b=b+1

NO es lo mismo hacer esto:

          Variable1 = 20;
          Variable2 = Variable1++;

          console.log(${Variable1} ${Variable2})

Resultado: 2021

Que hacer esto: 

          Variable1 = 20;
          Variable2 = ++Variable1;

          console.log(${Variable1} ${Variable2})

Resultado: 2121

1- Pues en el primer caso se realiza primero la asignación (Variable1=20) y después se realizala asignacion (Variable2= Variable1) y finalmente se incrementa (variable2 = Variable2 + 1)

2- Igualmente se realiza primero la asignación (Variable1=20) y después el incremento (Variable1= 20 + 1) y finalmente la asignacion (variable2 = Variable1)

Operador Condicional If

Este operador sirve para comprobar una condicion de alguna variable u objeto 

        if(condicion a cumplirse) {
          se ejecutará si se cumple la condicion }
        else {
          se ejecutará si no se cumple}

Ahora Veremos un Ejemplo

        const valorPasaje = 1000;

        if (valorPasaje === 1000) {
          console.log('El pasaje vale 1000');
        }

Resultado: En esta ocacion la condicion es verdadera por lo cual se mostrara el mensaje

Condicional Ternario

También tenemos el condicional ternario para simplificar la sintaxis de este operador en una linea

        (Expresión) ? (Si es true): (Si es false)

        (x>y) ? alert("el mayor es x") : alert("el mayor es y")

Condicional Switch

Esta condicional permite verificar una sentencias varias veces seguidas

        switch(condicion) {
          case condicion a cumplirse 1:
            se ejecuta para este caso 1;
          case condicion a cumplirse 2:
            se ejecuta para este caso 2;
          case condicion a cumplirse 3:
            se ejecuta para este caso 3;
            break;
          default :
            se ejecutará en caso de no cumplirse ningun caso;
            break;
        }

Aca un ejemplo facil para comprobar el valor del pasaje a una ciudad 

        let ciudadDestino = "Lima"

        switch(ciudadDestino) {
          case "Bogotá": 
              valorPasaje = 500;
              break;
          case "Lima": 
              valorPasaje = 400;
              break;
          case "Santiago": 
              valorPasaje = 380;
              break;
          case "Montevideo": 
              valorPasaje = 200;
              break;
          default:
              console.log(`No hay pasajes para la ciudad indicada`);
              break;
          }

Resultado: valorPasaje = 400

Bucle For

Sirve para recorrer un Array

        for([Inicialización]; [Condición]; [Expresión de actualización])
          {
            Instrucciones a repetir
          }

Ejemplo y el resultado 

          var MaxNumero = 10 ;
          
          for(i = 1; i <= MaxNumero; i++) {
            document.write( i )
          }

Resultado: 12345678910

Bucle For ... In

Produce una iteración a través de todas las propiedades de un objeto, Por ejemplo, este código recorre a document y nos muestra todas sus propiedades.

Sirve para recorer listas 

          for(var 'Propiedad' in 'Objeto') {
            Instrucciones a repetir
          }

Ejemplo y el resultado 

            const personas = ["alan","memo","lisa"]
            for(var i in personas) {
              document.write(personas[i]);
            }

Resultado: alan memo lisa

Bucle While

La sintaxis del bucle while 

        while(Condición) {
          Instrucciones a repetir
          }
      
        const numero=10;
        let contador = 0;

        while (numero <= contador) {
          console.log(contador)
          contador++
        }

Bucle Do ... While

La sintaxis del bucle Do ... while 

            Do {
              Instrucciones a Realizar
            } While (Condicion a cumplir);
          
            const numero=10;
            let contador = 0;
    
            do{
              console.log(contador)
              contador++
            } While(numero <= contador)

La construccion con With

Este nos ahorra escribir siempre el nombre de un objeto si vamos a utilizarlo muchas veces para acceder a sus métodos, propiedades,... 

        with(objeto) {
          Varias sentencias
        }

Por ejemplo 

        with(document) {
          write('Esta sentencia evita escribir el metodo "write"');
          write('cuando usas muy seguido. Texto ');
          write('de prueba.');

Resultado: Esta sentecia evita escribir el metoso "write" cuando usas muy seguido. Texto de prueba
